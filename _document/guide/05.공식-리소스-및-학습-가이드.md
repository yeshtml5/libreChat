## 🎓 학습 로드맵

### **초보자 코스**
```
1주차: 기본 설치 및 설정
├── Docker 설치 및 실행
├── 기본 환경변수 설정
└── OpenAI API 연동

2주차: 기본 기능 활용
├── 채팅 인터페이스 사용법
├── 파일 업로드 기능
└── 다중 모델 사용법

3주차: 고급 기능 탐색
├── 에이전트 시스템 이해
├── RAG 기능 활용
└── 이미지 생성 기능
```

### **개발자 코스**
```
1주차: 개발 환경 구축

└── 코드 구조 이해

├── React 컴포넌트 구조 파악

4주차: 배포 및 운영
├── Docker 이미지 빌드
├── 프로덕션 배포
└── 모니터링 및 로깅
```

## 📖 추천 학습 순서

### **1. 기본 이해 단계**
1. **공식 홈페이지 탐방** - 프로젝트 개요 이해
2. **설치 가이드 따라하기** - Docker로 빠른 체험
3. **기본 기능 테스트** - 채팅, 파일 업로드 등

### **2. 활용 단계**
1. **AI 엔드포인트 설정** - 다양한 AI 모델 연결
2. **고급 기능 체험** - 에이전트, RAG 등
3. **커스터마이징** - 인터페이스, 설정 조정

### **3. 개발 단계** (개발자용)
1. **로컬 개발환경 구축** - 소스 코드 빌드
2. **코드 구조 분석** - 아키텍처 이해
3. **기능 추가/수정** - 실제 개발 경험

### **4. 배포 및 운영 단계**
1. **프로덕션 배포** - 실서버 환경 구축
2. **모니터링 설정** - 로그, 메트릭 수집
3. **스케일링** - 성능 최적화
